## 前言

我前段时间打算给我的博客网站增加一些功能，例如 GitHub 登录、Cloudflare 全家桶、SEO 优化什么的，很可惜。都因为我没有域名，总是因为一些原因而失败。于是我想了想，一年几十块买个域名，也不算贵，因为我之前也搞过，只不过不是我出钱。



## 为什么选择 [Cloudflare](https://www.cloudflare.com/) 注册域名？

[Cloudflare Registrar](https://www.cloudflare.com/products/registrar/) 最大的特点是**不赚差价**。它不像传统注册商那样靠域名盈利，而是以此作为进入其生态系统的入口。

- **真正的批发价**：Cloudflare 承诺永久以**成本价**销售域名。你支付的每一分钱都是直接交给注册局，CF 不加收任何服务费，且续费价格固定，没有“首年低价、次年暴涨”的套路。
- **免费隐私保护**：默认提供 WHOIS 隐私保护，隐藏你的姓名、邮箱和地址。而在其他平台，这可能是一项年费支出。
- **顶级安全防护**：免费赠送企业级 **DNSSEC** 防止域名劫持，并支持硬件密钥（如 YubiKey）登录，安全性拉满。
- **管理一步到位**：如果你打算用 Cloudflare 部署博客，域名在这里买可以省去修改 DNS 的等待时间，证书申请和解析都是秒级自动完成。

所以在 Cloudflare 购买域名算是批发价格了，我下面提供一个表格，来对比一下 Cloudflare 和阿里云、腾讯云之类的区别。

| **维度**     | **Cloudflare**           | **传统云厂商 (阿里/腾讯/Namecheap)** |
| ------------ | ------------------------ | ------------------------------------ |
| **定价**     | 成本价，无溢价           | 市场价，首年优惠次年贵               |
| **隐私保护** | 默认免费开启             | 免费或需额外购买                     |
| **管理体验** | 开发者友好，集成度极高   | 功能分散，配置较多                   |
| **备案支持** | **不支持**国内工信部备案 | **支持**国内工信部备案               |

Cloudflare唯一存在一点不足，那就是不支持国内工信部备案，所以根据你的需求，是否一定要做工信部备案。对于我而言，我觉得目前不是特别有必要，我的服务托管到Cloudflare上了，唯一的支出也就域名了还能接受，并且白嫖Cloudflare的CDN国内访问也不是太大问题。如果我后续可能几年后有需要国内备案，那我就得再去整个国内服务器，还要买个国内域名了。

所以对于**不打算备案、使用 Cloudflare 生态**的个人博客来说，**CF 就是域名的终点站。**

## 准备工作

1. Cloudflare 账户
2. Visa 银行卡或者 PayPal

如果你还没有注册  [Cloudflare](https://www.cloudflare.com/) 账户，那就注册一个，Google 账号登录或者 GitHub 登录即可，点击该超链接。

对于 Visa 卡和 PayPal，对于国内读者可能有点难办了，因为需求原因，平时都没有用过。如果你找不到个哥们什么的帮你代付，你只能自己想办法了，很多银行其实有提供给学生的支持 Visa 的信用卡，以下内容我以工商银行的星座卡学生办举例（没有支付问题的朋友可以自行跳过，并非广告，解决国内学生朋友的问题）。



### 支付准备工作

---

**此处以工行星座卡校园版申请举例**

**申请条件**：需**本科以上**学历，学生卡尽量不超过2~3张（1+1套卡算1张），网贷（花呗，白条等）账户尽量不超过2**个**（其实一般你不乱搞什么的都有资质办理）

星座卡特点：

1. 银联（人民币）+Visa（外币）套卡，卡等级为普卡，有效期内免年费（约等于永免）。

2. 绝大部分地区的校园版是**0额度**！非校园版需有实习/工作单位。
3. **11币种账户**，但对征信无碍，放心申。

#### *申请步骤：*

1. 打开中国工商银行app，点击顶部的搜索框输入星座卡学生办
2. 在页面中产品一栏，选择宇宙星座卡校园版（VISA+银联）
3. 按照流程填写信息，以及信用卡的邮寄地址
4. 等到银行卡拿到后，你就可以去线下网点激活了

当然你也可以直接去线下网点办理，不过要注意不要异地办理，例如你可能填写的注册银行是你家那边的支行，但是你在外省读大学，可能到了外省的线下网点就无法激活了，所以还是建议选一个你方便激活的银行地址。

#### 转换美元

1. 打开中国工商银行app，点击顶部的搜索框输入美元兑换
2. 在产品栏看到结售汇，点击选择购汇
3. 币种选择美元然后按照你的需求购买
4. 受到短信后检查一下对应银行卡的美元余额

记住不要美元余额刚刚好是你的域名价格，因为境外网站可能对支付会有最低额度的设置。

我还是很建议你办一张Visa卡，总有用到的时候。

## Cloudflare 注册域名步骤

点击进入以下网址 [https://www.cloudflare.com/zh-cn/products/registrar/](https://www.cloudflare.com/zh-cn/products/registrar/)，你可以看一下 Cloudflare 的域名注册介绍，这个网址也很容易搜到，在搜索引擎搜查 Cloudflare 域名购买就可以直达。你也可以选择左侧导航面板点击： 域注册 → 注册域，或者直接点击链接访问：[https://dash.cloudflare.com/domains](https://dash.cloudflare.com/domains)（右上角可以选择切换语言）

![](https://img.danarnoux.com/posts/how-to-register-a-cloudflare-domain/step1.png)

搜索你需要的域名，我们以danarnoux为例，可以看到有很多后缀的域名

![](https://img.danarnoux.com/posts/how-to-register-a-cloudflare-domain/step2.png)

至于这些域名后缀代表什么，我为您提供一份表格进行参考

| **后缀**  | **类型** | **建议使用场景**                   | **特点**                                            |
| --------- | -------- | ---------------------------------- | --------------------------------------------------- |
| **.com**  | 通用     | **首选**。个人博客、品牌、商业     | 全球认可度最高，用户心智首选，利于 SEO。            |
| **.net**  | 通用     | 网络服务、技术博客、替代方案       | 历史悠久，专业感强，常作为 .com 被占用后的备份。    |
| **.org**  | 非营利   | 开源项目、社区、公益组织           | 带有信任感和权威性，适合非营利性技术社区。          |
| **.io**   | 技术     | **技术博客**、开发者工具、初创公司 | 在程序员圈子极受欢迎，代表 Input/Output，极简且酷。 |
| **.me**   | 个人     | **个人简历**、生活随笔、作品集     | 个性化极强，非常适合强调“自我”品牌的个人博客。      |
| **.info** | 信息     | 知识库、资讯网站、说明文档         | 价格通常较便宜，适合做纯信息展示类项目。            |
| **.dev**  | 开发者   | 编程教程、代码库、开发日志         | 谷歌推出的后缀，强制 HTTPS 安全加密，极具辨识度。   |

你可能会注意到.uk的价格比.com接近便宜一倍，那我是不是可以购买这个？我的建议是，**这个便宜还是不贪为妙**

**.uk** 是英国（United Kingdom）的国家代码顶级域名（ccTLD）。它由 Nominet 机构管理，类似于中国的 `.cn` 或美国的 `.us`。可以购买，但有“**英国地址**”要求。如果你没有英国地址，很多注册商（包括 Cloudflare）可能会拒绝你注册，或者要求你使用代理服务。如果你随意填写虚假的英国地址，一旦被管理机构抽查，域名可能会被直接封禁（Suspended）。

权衡一下，选择你想要的域名即可。记得注意一下有无被注册以及**续费价格**。

## 购买 Cloudflare 域名

1. 选择好域名后，点击Confirm确认
2. **按年计费**，选择你要购买的年份，注意选择支付方式后的 **Automatic renewal**（自动续期），选择是否勾选

   ![](https://img.danarnoux.com/posts/how-to-register-a-cloudflare-domain/step3.png)
3. 填写相关信息，以及发票信息
4. 选择银行卡支付的时候，一定要注意不要输错安全码，输错三次银行卡会被锁，后续只能挂失新开一张银行卡了。（银行卡）
5. 如果你支付的时候出现银行卡被拒绝的情况，检查一下账户内美元余额，安全码是否输错，卡号是否输对，实在不行可以联系银行客服。在我支付的时候，还出现了一种问题，估计是地理位置和ip不一致被支付网关拒绝了，说人话就是我停止使用香港节点进行支付成功了。如果你也出现了这种情况，可以试试，只是支付会慢一点。
6. 支付成功后，检查邮箱的发票以及银行卡余额。
7. 去左侧仪表盘内点击Domains，检查是否存在购买的域名。

## 域名解析与 DNS 设置

点击左侧面板的 **Domains**，点击右侧的三个点进入配置页面配置 DNS。

![](https://img.danarnoux.com/posts/how-to-register-a-cloudflare-domain/step4.png)

可以参照我的设置选择 CNAME 进行解析，打码的内容为不重要，**Cloudflare Proxy**（即橙色云）可默认勾选，提升网站速度和安全性。设置完后记得保存

![](https://img.danarnoux.com/posts/how-to-register-a-cloudflare-domain/step5.png)
